# WashSetu MVP Wireframe Blueprint

## Customer App Screens
1. Splash + OTP Login
2. Nearby Vendors List (filters + sorting)
3. Vendor Detail (services + ratings + slot CTA)
4. Slot Selection (date/time + capacity state)
5. Checkout (address + payment method)
6. Booking Confirmation
7. Live Tracking Timeline
8. Review Submission

## Partner App Screens
1. Partner Login
2. Dashboard (today metrics + queue)
3. Slot Management (capacity per slot)
4. Booking Detail + Status Update
5. Payout Summary

## Admin Dashboard Screens
1. Vendor Onboarding Queue
2. Booking Ops Dashboard
3. Incident/Dispute Management
4. Promotion Campaign Controls

## UX Notes
- Keep booking completion under 6 user actions after login.
- Use status chips consistently across user roles.
- Surface failure states with clear retry actions.
- Keep primary CTA fixed on bottom for mobile screens.
